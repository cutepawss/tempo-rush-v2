**Tempo Rush** is a high performance Web3 "Capture the Flag" protocol built on the **Tempo Testnet**. It challenges operators to engage in a relentless battle for network supremacy, leveraging Tempo's sub second finality and native stablecoin architecture.

---

## MISSION OBJECTIVE

The protocol operates on a competitive dominance model:
1. **Capture:** Seize the digital flag from the current bearer.
2. **Hold:** Maintain your ground against rival operators.
3. **Ascend:** Be the active bearer at **00:00 UTC** to trigger a snapshot and secure an exclusive **Speed Demon NFT Artifact**.

---

## TECH STACK & ARCHITECTURE

Designed for zero latency interactions and a retro tactical user experience:

* **Core Logic:** Solidity Smart Contracts (EVM Compatible).
* **Frontend:** React 18 + Vite (Optimized for maximum HMR performance).
* **Web3 Integration:** Ethers.js (v6) for seamless blockchain uplink.
* **Network:** Tempo Testnet (Built by Stripe & Paradigm).
* **Styling:** Custom Retro Cyberpunk CSS Framework with 60FPS scanline animations.
